# HEALTHCARE_PLATFORM

Simple full-stack app to upload, list, download and delete PDF medical documents.

## Tech
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: MySQL
- File storage: local `backend/uploads/`

## Setup (local)
Prereqs: Node.js (16+), npm, MySQL server.
